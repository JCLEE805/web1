HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. HTML describes the structure of a web page semantically and originally included cues fosr the appearance of the documents.



HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms.
